# ntheme

Theme resolver for Nette — active/fallback template resolution and asset paths.

## Key Classes
- `ThemeResolver` — resolves template paths based on active theme

## Conventions
- PHP 8.2+ with `declare(strict_types=1)`
- PSR-4 autoloading in `src/`
- Nette DI Extension for service registration
- PHPStan level 9
- Part of `jansuchanek/*` on Packagist
- GitHub: https://github.com/JanSuchanek/ntheme
